# Contributing to ZamPOS

Thank you for your interest in contributing! ZamPOS is a community project and welcomes contributions from developers at all levels.

## How to Contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature-name`
3. Commit your changes: `git commit -m "feat: description of your change"`
4. Push to your fork: `git push origin feature/your-feature-name`
5. Open a Pull Request

## Areas We Need Help With

- **Translations** — Nyanja, Bemba, Tonga, Swahili
- **Testing** — especially on low-end Android devices
- **UX feedback** — from actual informal market traders
- **Documentation** — tutorials and guides for new developers

## Code Style

- Frontend: ESLint + Prettier (config included)
- Backend: Black formatter for Python
- Commit messages: follow [Conventional Commits](https://www.conventionalcommits.org/)

## Questions?

Open an issue or reach out via GitHub Discussions.
